# FitnessTracker
Трекер упражнений. 7 основных экранов и 3 стартовых экрана.
Работа с:
- TableView(customTableViewCell)
- CollectionView(customCollectionViewCell)
- ScrollView
- AlertControll(customAlert)
- DatePicker
- UIKit
- Notifications
- Basic CoreAnimation
- Passing data between ViewControllers
- JSON
- Realm
- TabBarController
- No StoryBoard
